# src/ts/ - TypeScript Source

Root of the frontend TypeScript/React codebase for Anytype desktop.

## Directory Structure

| Directory | Purpose |
|-----------|---------|
| `component/` | React UI components (blocks, pages, menus, popups, sidebars, widgets, cells, comments, etc.) |
| `lib/` | Core libraries: API layer, constants, services, utilities, web mode |
| `store/` | MobX state management stores (auth, blocks, details, menus, popups, chat, comments, etc.) |
| `model/` | Data model classes for blocks, views, filters, notifications, chat/comment messages |
| `interface/` | TypeScript interfaces and enums for the entire app |
| `hook/` | Custom React hooks (currently empty barrel) |
| `docs/` | In-app documentation (What's New content, onboarding) |
| `workers/` | Web Worker entry points (PixiJS bundle for graph) |
| `test/` | Test setup (`setup.ts` for test environment) |

## Entry Points

- `entry.tsx` — Electron app bootstrap (renders `<App />` inside `<ErrorBoundary>`)
- `entry.web.tsx` — Web/browser bootstrap (sets up Electron mock, server config, then dynamic-imports App)
- `app.tsx` — Main React application component

## Type Declarations

- `global.d.ts` — Declares build-time globals (`SPARK_ONBOARDING_URL`, `SENTRY_DSN`, `__IS_EXTENSION__`, etc.) and module declarations
- `auto-imports.d.ts` — Generated by `unplugin-auto-import`; declares all auto-imported globals (`S`, `U`, `C`, `J`, `H`, `keyboard`, `translate`, `dispatcher`, `analytics`, `sidebar`, `Action`, `Dataview`, `Mark`, `Relation`, `Renderer`, `Sound`, `Survey`, `Preview`, `Onboarding`, `scrollOnMove`, etc.)

## Key Patterns

- **MobX** for reactive state management (`observer()` HOC on components)
- **gRPC** communication with middleware via `C.*` commands
- **Block-based** document model (all content is composed of blocks)
- **`translate()`** for all UI text (i18n via `src/json/text.json`)
- **Auto-imports** — common globals are auto-imported and available without explicit import statements

## Import Aliases

- `I` — Interfaces (`src/ts/interface/`)
- `C` — gRPC Commands (`src/ts/lib/api/command.ts`)
- `S` — Stores (`src/ts/store/`)
- `U` — Utilities (`src/ts/lib/util/`)
- `J` — JSON constants (`src/json/`)
- `H` — Hooks (`src/ts/hook/`)

## Component Subdirectories

`component/` contains 19 subdirectories: `block`, `cell`, `comment`, `drag`, `editor`, `footer`, `form`, `graph`, `header`, `list`, `menu`, `notification`, `page`, `popup`, `preview`, `selection`, `sidebar`, `util`, `widget`.

## Library Subdirectories

`lib/` contains: `api/`, `constant/`, `service/`, `util/`, `web/`, and top-level modules: `action.ts`, `analytics.ts`, `animation.ts`, `dataview.ts`, `focus.ts`, `highlight.ts`, `history.ts`, `keyboard.ts`, `mark.ts`, `onboarding.ts`, `preview.ts`, `reactionScheduler.ts`, `relation.ts`, `renderer.ts`, `scrollOnMove.ts`, `sidebar.ts`, `sound.ts`, `storage.ts`, `survey.ts`, `translate.ts`.
