# Claude memory snapshot

This folder is a snapshot of the [Claude Code](https://claude.com/claude-code)
project memory captured while setting up and running this repository. Each `.md`
file is one durable fact (how to run, how the toolchain was set up, where the
vault lives, hosting constraints); `MEMORY.md` is the index.

These notes are machine/environment-specific (local paths, account identifiers)
— treat them as setup history for this checkout, not as authoritative docs. The
maintained guide is [`../SETUP_WITH_CLAUDE.md`](../SETUP_WITH_CLAUDE.md).
